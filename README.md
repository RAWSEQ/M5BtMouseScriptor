# M5BtMouseController

## 概要

iPhoneのリモートマクロ操作を実現する為のコントローラーです。

[M5BtMouseController](https://github.com/RAWSEQ/M5BtMouseController) と組み合わせて利用します。

![img](https://ltside.com/images/m5btmouse.drawio.png)

**↑上記図の `PC` のプログラムです。**

## 環境

- Node v14.x.x
- (動作確認したのはWindowsのみ）

## インストール方法

node 上記環境にてコマンドを実行

```
node i
```
